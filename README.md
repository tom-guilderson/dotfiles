## To Setup emacs on a new machine, assuming emacs 29+

When you run emacs for the first time, you will get errors about missing packages, so install the following packages with package-install command in emacs:

`ESC-x package-install` `go-mode`

`ESC-x package-install` `go-dlv`

`ESC-x package-install` `rust-mode`
